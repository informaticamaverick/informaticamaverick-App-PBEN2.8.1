# 🚀 Guía Rápida: Inicializar Firebase

## ✅ PASOS PARA POBLAR FIREBASE:

### 1️⃣ Abrir la Pantalla de Admin
1. Abre la app
2. En la esquina superior derecha, toca tu **avatar/inicial**
3. En el menú desplegable, selecciona **"🔥 Inicializar Firebase"**

### 2️⃣ Crear las Categorías
1. En la pantalla de admin, presiona el botón verde:
   ```
   1️⃣ Crear 18 Categorías
   ```
2. Espera a que termine (verás un mensaje de éxito)
3. El contador mostrará: "Categorías en Firebase: 18"

### 3️⃣ Crear las Subcategorías de Mecánico
1. Ahora presiona el botón azul:
   ```
   2️⃣ Crear Subcategorías Mecánico
   ```
2. Espera a que termine
3. Verás un mensaje: "🎉 ¡Todo Listo!"

### 4️⃣ Volver al Dashboard
1. Presiona el botón **"✅ Volver al Dashboard"**
2. ¡Las categorías ya deberían estar cargándose desde Firebase!

## ⚠️ IMPORTANTE:

### ⚡ Solo usa esto UNA VEZ
- Estos botones son **solo para inicializar** Firebase

### ✅ Después de Inicializar:
- Puedes agregar más categorías desde **Firebase Console**
- Puedes editar las existentes desde **Firebase Console**
- No necesitas volver a esta pantalla

2. Selecciona tu proyecto
